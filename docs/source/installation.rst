============
Installation
============


Stable release
--------------

To install `paniz_image`, run this command in your terminal:

.. code-block:: console

    $ pip install paniz_image

This is the preferred method to install `paniz_image`, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for `paniz_image` can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/paradise1260/paniz_image

Or download the `tarball`_:

.. code-block:: console

    $ curl  -OL https://github.com/paradise1260/paniz_image/tarball/main

Once you have a copy of the source, you can install it. The method of installation will depend on the packaging library being used.

For example, if `setuptools` is being used (a setup.py file is present), install `paniz_image` with:

.. code-block:: console

    $ python setup.py install

If `poetry` is being used (poetry.lock and pyproject.toml files are present), install paniz_image with:

.. code-block:: console

    $ poetry install


.. _Github repo: https://github.com/paradise1260/paniz_image
.. _tarball: https://github.com/paradise1260/paniz_image/tarball/master
